# sgr-ksmt's website

- https://sgr-ksmt.dev
- Built with Flutter
- Running on Firebase Hosting

## Development

```bash
$ git clone ...
$ cd path/to/repository
$ fvm install
$ fvm flutter pub get
$ fvm flutter run -d chrome --web-renderer=html
```

## Deployment

```bash
$ cd path/to/repository
$ yarn deploy
```
